# inline::prompt-guard

## Description

Prompt Guard safety provider for detecting and filtering unsafe prompts and content.

## Configuration

| Field | Type | Required | Default | Description |
|-------|------|----------|---------|-------------|
| `guard_type` | `<class 'str'>` | No | injection |  |
